## Журнал изменений

Для просмотра истории коммитов используется команда _git log_. Для этого в терминале с папкой-репозиторием _git log_. В показанном журнале обязательно будут:

- Хеш(номер) коммита
- Дата и время коммита
- Автор коммита
- Текст сообщения к коммиту
